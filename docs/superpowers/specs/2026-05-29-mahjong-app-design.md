# 麻将小账本 — 设计说明书

**日期：** 2026-05-29  
**用户：** 杨昊翔（小翔子）→ 妈妈  
**平台：** 移动端 PWA（单文件 HTML）

---

## 一、产品概述

一个面向妈妈辈用户的麻将记账 PWA 应用。每日记录净输赢金额，自动以儿子（小翔子）的口吻生成一句俏皮暖心的鼓励话。胭脂玫红配色，仪表盘式布局。

**核心价值：** 让妈妈每天记账时感受到儿子的关心，输赢都有一个好心情。

---

## 二、技术方案

| 项 | 选择 |
|---|---|
| 形态 | 单文件 HTML + 内联 CSS/JS |
| 安装 | PWA（manifest + Service Worker），可加桌面 |
| 存储 | localStorage |
| 导出 | 生成文本文件下载 |
| 依赖 | 无 |

## 四、界面结构

### 首页（仪表盘）

- **顶栏**：胭脂渐变背景，白色文字「🏮 麻将小账本」+ 当日日期
- **大数字区**：今日净胜金额，72px 粗体 — 赢显示绿色「+180」，输显示红色「-120」，未记录显示灰色「—」
- **副标题**：如「今日净胜 ¥180 · 盈利日 ✨」
- **消息卡片**：淡粉渐变圆角卡片，大表情 + 一句儿子口吻的话
- **统计栏**：三格并排 — 本周净胜 / 本月净胜 / 胜率
- **主按钮**：「✏️ 记录今天」（圆角大按钮，玫红，带阴影）
- **底部 Tab**：首页 🏠 | 历史 📋 | 导出 📤

### 历史页

- 按日期倒序的卡片列表
- 每行显示：日期 + 输赢金额（绿色/红色）+ 展开箭头
- 点击展开：显示当日的消息内容
- 空状态：「还没有记录哦~」

### 导出页

- 文本预览区（白色圆角卡片）
- 「复制文本」按钮
- 「下载文件」按钮
- 导出格式示例：

```
🏮 麻将小账本

2026-05-29  赢 ¥180  |  妈，今天赢180，比我写代码赚得还稳！
2026-05-28  输 ¥120  |  妈，输了别放心上，明天肯定翻盘！
```

## 五、记账交互流程

1. 首页点「✏️ 记录今天」
2. 弹出底部面板（Sheet），包含：
   - 两个大按钮：「🀄 赢了」/「🀄 输了」—— 二选一高亮
   - 数字键盘（0-9 + 退格 + 清除）—— 输入金额
   - 「保存」按钮（玫红圆角大按钮）
3. 保存后自动关闭面板，首页刷新显示今日数据 + 消息

---

## 六、消息模板策略

5 档 × 每档 10+ 条模板，保存时当日随机择一。所有模板以「小翔子」视角书写。

| 档位 | 条件 | 风格 |
|---|---|---|
| 大赢 | ≥ 200 元 | 夸张骄傲 + 宠溺 |
| 小赢 | 1~199 元 | 开心肯定 + 俏皮 |
| 平 | = 0 元 | 轻松调侃 + 正能量 |
| 小输 | -199~-1 元 | 暖心安慰 + 幽默化解 |
| 大输 | ≤ -200 元 | 坚定支持 + 转移注意力 |

消息示例：
- 大赢：「妈，今天赢了{金额}块！比我写代码赚得还稳，要不咱家改行搓麻将吧😂 开心就好，别太累~」
- 小输：「妈，输了{金额}没事~ 这叫战略性休整，明天肯定翻盘！记得喝点热水早点休息❤️」

## 七、数据模型

```json
{
  "records": [
    {
      "date": "2026-05-29",
      "amount": 180,
      "message": "妈，今天赢180..."
    }
  ]
}
```

- `date`：ISO 日期字符串 `YYYY-MM-DD`
- `amount`：净输赢（正=赢，负=输，0=平）
- `message`：当日生成并保存的消息

**规则**：每天只保留一条记录。同一天再次保存则覆盖旧记录（方便妈妈记错了改）。

## 八、PWA 配置

- `manifest`：独立 `data:` URI 注入的 manifest，设置 `display: standalone`
- 图标：CSS 绘制简单麻将牌图标 + 生成 base64 PNG favicon
- Service Worker：缓存 HTML 自身，支持离线打开
- `theme-color`：`#BE185D`
